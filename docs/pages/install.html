Installation

Requirements

  1. Windows/Linux OS

  2. Python 3.9

  3. Installed modules (see requirements.txt):

    • xmlschema

    • defusedxml

    • reportlab

Installation steps

  1. Install Python 3.9

  2. Download kingfisher-main.zip

  3. Extract it wherever you want

  4. Change directory to extracted kingfisher-main

  5. Install all the required modules: python3.9 -m pip install -r requirements.txt

  6. Use scanner as it is described here
